print()での簡単な出力

Pythonのprint()関数を用いて本日の日付を出力します。 ただし、出力はページの一番下にされます。

from datetime import datetime now = datetime.now() print(f"今は{now:%Y年%m月%d日 %H:%M:%S}です。")

display()での出力

通常の出力

以下にdisplay()関数を用いて本日の日付を出力します。 出力もすぐ下にされます。

from datetime import datetime now = datetime.now() display(f"今は{now:%Y年%m月%d日 %H:%M:%S}です。")

指定した要素に出力

from datetime import datetime now = datetime.now() display(f"{now:%Y年%m月%d日 %H:%M:%S}", target="now")

ここから

  • 今は現在日時です。

ここまでの要素を指定して出力した結果が以下です。

  • 今は現在日時です。

改行を解消

from datetime import datetime now = datetime.now() display(f"{now:%Y年%m月%d日 %H:%M:%S}", target="now1", append=False)
  • 今は現在日時です。
